Features

Everything your terminal was missing.

Built for developers who want power without memorizing every flag — safely.

01

Natural Language

Describe what you want in plain English. ShellSpark figures out the exact command for your system.

02

Safety Engine

Every command is classified SAFE, WARNING, or BLOCKED before it ever runs. Dangerous commands are stopped automatically.

03

Dry-Run Default

Commands are shown but never executed unless you add --run. The default mode is always show-only.

04

Explain Mode

Add --explain to get a plain-English breakdown of what any generated command actually does.

05

Distro-Aware

Auto-detects your distro and package manager. Ubuntu, Fedora, Arch, Alpine, macOS — always the right command.

06

Activity Logging

Every command, its risk level, and exit code is logged to ~/.shellspark/logs/activity.log.

How it works

Five steps, zero surprises.

Every query goes through a clear pipeline before anything touches your system.

01

You type a natural language query

No flags, no man pages. Just describe what you want to do.

shellspark find files larger than 1GB
02

ShellSpark detects your system

Reads /etc/os-release to identify your distro and maps it to the right package manager — apt, dnf, pacman, brew, and more.

03

Groq AI generates the command

Your query + system context is sent to Groq's LLM. Only the query text is transmitted — no files, no credentials, no environment data.

04

Safety engine classifies the command

Before you see it, the command is analysed against structured patterns. Destructive operations are blocked. Risky ones require explicit confirmation.

✓ SAFE ⚠ WARNING — type 'yes' ✗ BLOCKED — never runs
05

You decide — run, explain, or cancel

Default shows the command only. Add --run to execute, or --explain to get a plain-English breakdown first.

Safety Engine

Know before you run.

Every generated command is classified before it reaches your terminal. BLOCKED commands never execute — ever.

find /home -name *.log -delete

Remote pipe to shell — confirm carefully
sudo shutdown -h now

rm -rf /

dd if=/dev/zero of=/dev/sda

:(){:|:&};

sx is the official short alias for shellspark. Both commands are identical — sx is provided as a convenience shorthand. Use whichever you prefer.

BLOCKED commands include: rm -rf on root/system dirs, dd to raw block devices, mkfs, fork bombs, disk wipers, and redirecting output to /dev/sda. WARNING commands (rm -rf on user dirs, curl | bash, shutdown, etc.) require typing 'yes' to confirm.
